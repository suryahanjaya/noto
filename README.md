## 🌟 Overview

**Noto** is a premium, modern notes application built with cutting-edge web technologies. Featuring a stunning liquid glass design inspired by Apple's iOS 16+ interface, it provides an intuitive and beautiful experience for managing personal notes with advanced organization features.

### ✨ Key Highlights

- 🎨 **Premium Liquid Glass Design** - Apple-inspired glassmorphism effects with backdrop blur
- 📱 **Fully Responsive** - Optimized for all devices from mobile to desktop
- ⚡ **Lightning Fast** - Built with Vite for optimal performance and instant hot reload
- 🌙 **Smart Dark Mode** - Automatic system theme detection with manual override
- 📂 **Advanced Organization** - Folder system, favorites, and archive functionality
- 🔍 **Intelligent Search** - Real-time note searching with advanced filtering
- 🎯 **Touch Optimized** - Mobile-first design with gesture support
- 💾 **Data Export** - JSON backup functionality for data portability

## 📦 Installation & Setup

### 📋 Prerequisites
- **Node.js** (v16 or higher) - JavaScript runtime environment
- **npm** or **yarn** - Package manager for dependencies
- **Git** - Version control system

### 🚀 Quick Start

1. **📥 Clone the repository**
   ```bash
   git clone https://github.com/suryahanjaya/noto.git
   cd noto
   ```

2. **📦 Install dependencies**
   ```bash
   npm install
   ```

3. **🔥 Start development server**
   ```bash
   npm run dev
   ```

4. **🌐 Open in browser**
   ```
   http://localhost:5173
   ```

### 🏗️ Build for Production

```bash
# Build optimized production bundle
npm run build

# Preview production build locally
npm run preview

# Deploy to Vercel (if using Vercel CLI)
vercel --prod
```

## 📊 Data Structure

### 📝 Note Object
```javascript
{
  id: number,           // Unique identifier (timestamp-based)
  title: string,        // Note title (max 50 characters)
  body: string,         // Note content (unlimited)
  archived: boolean,    // Archive status (true/false)
  createdAt: string,    // ISO date string for creation time
  folderId: number      // Associated folder ID (null for no folder)
}
```

### 📁 Folder Object
```javascript
{
  id: number,           // Unique identifier
  name: string,         // Folder display name
  date: string,         // Creation date (formatted)
  color: string,        // Folder color theme (blue/pink/yellow)
  noteCount: number     // Number of notes in folder (calculated)
}
```
